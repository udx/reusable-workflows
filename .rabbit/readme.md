# Generated Repo Context

This directory contains repo context generated by `dev.kit` for Rabbit CI and agent workflows.

- `.rabbit/context.yaml` is generated; do not edit it manually.
- Run `dev.kit repo` after changing workflow contracts, docs, examples, or other repo-owned context.
- Use the generated context to identify the repo refs, canonical commands, manifests, and repair gaps that Rabbit CI and agent workflows should consume.
